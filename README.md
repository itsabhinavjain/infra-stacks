# Infra for local development

## Purpose
To be able to quickly spin up various stacks for local development.

## Options
- Docker (Preferred)
	- Dockerfiles
	- Docker Compose Scripts
- Virtual machine 
- Installation on the local system 
	- For python always use virtual environments. Use `uv`
